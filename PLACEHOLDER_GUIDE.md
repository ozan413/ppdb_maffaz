# Daftar Placeholder untuk Template DOCX

Template DOCX di `storage/app/templates/Alur PSB I'dad Tahfidz.docx` dapat menggunakan placeholder berikut dengan format `${nama_placeholder}`.

## Data Pendaftaran

| Placeholder           | Deskripsi           | Contoh                         |
| --------------------- | ------------------- | ------------------------------ |
| `${nama_akun}`        | Nama akun user      | John Doe                       |
| `${email}`            | Email user          | john@example.com               |
| `${program}`          | Nama program        | Iddah Tahfidz                  |
| `${tanggal_daftar}`   | Tanggal pendaftaran | 06 Januari 2026                |
| `${status_kelulusan}` | Status kelulusan    | LULUS / TIDAK LULUS / MENUNGGU |

## Data Diri Santri

| Placeholder                         | Deskripsi           | Contoh                |
| ----------------------------------- | ------------------- | --------------------- |
| `${nama_lengkap}`                   | Nama lengkap santri | Ahmad Fauzan          |
| `${nama_panggilan}`                 | Nama panggilan      | Fauzan                |
| `${tempat_lahir}`                   | Tempat lahir        | Jakarta               |
| `${tanggal_lahir}`                  | Tanggal lahir       | 15 Maret 2010         |
| `${usia}`                           | Usia santri         | 15                    |
| `${gender}` atau `${jenis_kelamin}` | Jenis kelamin       | Laki-laki / Perempuan |
| `${no_hp}`                          | Nomor HP santri     | 08123456789           |
| `${email_santri}`                   | Email santri        | ahmad@example.com     |

## Cara Menggunakan

1. Buka file template DOCX di Microsoft Word atau Google Docs
2. Tambahkan placeholder dengan format `${nama_placeholder}`
3. Contoh: "Nama Lengkap: `${nama_lengkap}`"
4. Simpan template
5. Saat download dari sistem, semua placeholder akan otomatis diganti dengan data santri dari database

## Catatan

-   Jika data tidak ada di database, placeholder akan diganti dengan tanda `-`
-   Gunakan nama placeholder persis seperti di tabel di atas
-   Huruf kecil semua dan gunakan underscore untuk spasi
